Align is distributed as a zip archive containing Align.jar and the file conf.align, which must be present in the directory from which Align is run. Unzipping this archive will create the folder align containing these two files. Launch Align.jar in the usual way (i.e. either double-click or use the command line java -jar Align.jar).

System Requirements Align requires Java 1.5 or later. It is otherwise self-contained.
