Preset Scoring
Permits the user to select one of the preset scoring matrices (DNA Blast or Transition Transversion)

Manual Scoring
With manual scoring, the user selects the alphabet (either a standard biological alphabet or one specified directly) and chooses either simple or matrix scoring. With simple scoring the user must further specify origination and gap penalties and match and mismatch scores. With matrix scoring the user must further specify a scoring matrix.
See Scoring for details.

Animation On
When selected causes the program to start up in Animation On mode.
Animation Speed
Sets the initial animation speed.
See Operations for details.
