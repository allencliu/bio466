File Menu
Load Sequences ...
Brings up the "Load Sequences" dialog. Sequences can be typed directly into the dialog, or the user can browse for a file containing two sequences in Fasta format.
Save Sequences ...
Allows the user to save the current sequence pair in Fasta format.
Load Configuration ...
Allows the use to load a previously saved scoring configuration.
Save Configuration ...
Allows the use to save a scoring configuration.
Save and Exit
Exits the program, saving the current sequence pair. This pair will be automatically loaded when the program is run again.
Quit
Exits without saving the current sequence pair. Equivalent to using the frame Close box.
