Initialization
  • Select algorithm of interest using Algorithm radio button.
  • Click Init to initialize matrix.
Manual mode
In Manual mode the user performs the alignment by adding or deleting gaps in the sequences. Clicking Evaluate causes the program to score the alignment.

Right-clicking on the sequence inserts a gap at the click-point, while left-clicking closes the nearest gap to the left of the click-point. Clicking Reset re-initializes the sequence.

Automatic mode
In Automatic mode the program performs the alignment using the selected algorithm. When Animation On is selected, each step is illustrated. The animation speed is determined by the Speed slider. When the algorithm is complete the user may click on a table slot to see how the algorithm chose a value for that slot.

If the user previously performed a manual alignment, comparison can be made between the manual and automatic alignments by clicking the radio buttons back and forth between the two modes.
